## Prerequisites

* PostgreSql

## Installation

```sh
mvn clean package

java -jar .target\LibraryAutomation-0.0.1-SNAPSHOT.jar
```

## Used Technologies

* Spring Boot 2.5.6
* Spring Boot Jpa
* PostgreSql
* Lombok
* Model Mapper
* OpenApi
